# VeriLogos Publication-Ready Package

This frozen package contains the publication-ready VeriLogos reporting artifacts, audit evidence, manuscript tables, figures, and reproducibility references assembled from existing project outputs.

Important points:
- This is not a new experimental run.
- No training, evaluation, rerun, or tuning was performed during packaging.
- Audit-only review has already been completed.
- Scientific claims must remain conservative and consistent with the included audit documents.
- Collapsed/problematic runs remain documented exactly as frozen and were not altered.
